Photo Gallery For Flickr, an improved WordPress plugin built to display your flickr photo album gallery in your website through very easy steps. It allows to showcase your flickr photo gallery on your website with an improved layout.

== Description ==

Photo Gallery For Flickr is based on Flickr API. It is used to display and manage your flickr photo gallery in your WordPress published page/post.

Photo Gallery For Flickr plugin shows your Flickr images in your WordPress website with a attractive showcase and transitions that gives a soothing effect to your visitors. We have invested a lot of efforts and resources in building this plugin, which is reflected through our customer satisfaction and positive feedback for photo gallery for flickr plugin.

**Photo Gallery For Flickr Features**

* **Responsive Design** - You can view gallery into all major portable devices like large desktop, laptops, tabs, mobiles through flickr plugin for wordpress.

* **Blueimp Lightbox Image Slider Preview With Touch Support** - You can see all gallery image large preview into Blueimp lightbox with touch sliding feature.

* **PhotoBox Image Slider Preview With Touch Support** - You can see all gallery image large preview into PhotoBox lightbox with touch sliding feature.

* **Simple & Very Easy Admin Gallery Dashboard** - Plugin admin dashboard is very simple with one page settings only, it's not with lots of settings and configurations.

* **All Major & Latest Browser Compatible** - Google Chrome, Mozilla Firefox, Internet Explorer.

* **All Major Device Compatible** - Compatibility is tested with Phone, iPad, Tablets, and PC.

* **Image Fetch Limit 200** - You can fetch 200 images per Photo Gallery For Flickr shortcode into free version of plugin.

* **Photo Gallery For Flickr Shortcodes** - You can use shortcode to display gallery output on any Pages / Posts.

* **Photo Gallery For Flickr Widget** - Plugin also provide a widget to display gallery into theme supported sidebar and widgets area.

* **Multilingual Translation Ready** - Translate plugin settings panel into any language you want. If you not familiar with plugin translation you can visit us at weblizar.com and contact us.

== Installation ==

**Photo Gallery For Flickr [Installation Guide]**

1. You can:
 * Upload the 'photo-gallery-for-flickr' folder to '/wp-content/plugins/' directory via FTP.
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **Photo Gallery For Flickr** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Ready, now you can enjoy it, and if you like it and find it useful.

**How to use Photo Gallery For Flickr?**

1. Download the plugin using WordPress.org or you can download the plugin through WordPress admin dashboard 'Plugins -> Add New'
2. After download and activate plugin, go to 'Photo Gallery For Flickr -> Add New Gallery' sub menu page
3. Type the 'Title' for the gallery, provide your 'Flickr API Key', provide your 'Flickr Album ID', Custom CSS field is optional to use, if you want to add some custom style then you use custom css setting.
4. Hit the publish button. Below the publish button, copy the photo gallery for flickr shortcode *[PGF id=123]*, now you can publish photo album into any Page or Post using this shortcode.
